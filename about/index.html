About Me

地球年公元 2007 出生，一个平平凡凡，很容易哭的学生。

五年级时，一段奇妙的经历让我学会了如何出海航行，看到之前从未见过的世界。从最开始连 GitHub 是什么都不知道，我逐渐掌握了一些 Python 基础知识，能写出来一个丑陋不堪的图书管理系统，记得它甚至没有图形化界面，直接跑在命令行。后来熟悉了这些，趁着 Black Friday 租了一年的服务器，拥有了一部属于自己的 VPS。通过 ssh 与一台远在洛杉矶的 Linux 主机建立联系，看着终端里一闪一闪的光标，让年少的我兴奋无比。那以后不断折腾，不断折腾，变成了今天的 Koslin。

学生的天职是学习，目前为止我觉得自己做的不错。不展开细说，我不希望自己能拿出手的，只有一张张奖牌和成绩单，而是丰富的经历和颇有深度的内涵。

最近正在探索 SwiftUI 的开发，未来这一年想做出一个不错的测试应用，目前还没什么头绪。

Hobbies

除了喜欢在电脑前洞察世界（折腾），我还有蛮多爱好。

我喜欢书写，用钢笔。圆珠笔让我厌烦写字的感觉，因为太过于平滑，显得机械死板而没有生意。钢笔刮纸的那种触感很打动我，我也十分喜欢偶尔笔尖沾上一些毛写出带有艺术感的「毛笔字」。有几个活页本在学校，用来写文章、记录生活，那也是实体版本的 「Koslin’s Blog」。

热爱跑步。每次穿上心爱的跑鞋，站在跑道上，我只想更快。初中两次运动会，都是年级上个人得分榜的前三，这让我倍感骄傲。

偶尔游戏。有一部 Switch 和一台 Xbox，喜欢玩《塞尔达传说：旷野之息》，《极限竞速：地平线5》还有 《FIFA》系列。主机上的游戏有些昂贵，却总比浮躁的手游更能让我享受 Gaming。

总会听歌。一部 AirPods Pro 和 SONY XM 5，还有被岁月留下道道印痕的 Walkman，是我对音乐的执着。最近喜欢 Lana Del Rey，Taylor Swift 还有周董。我拥有一个 20G 的音乐库，都是我的心血。

果粉。我很小时候就拥有了一部 iPod Touch，经常用它开卡丁车还有切水果。在那以后买齐了 Apple 全家桶，并且订阅着 Apple TV+ 还有 Apple Music。我喜欢那句「Designed by Apple In California」，那是一种对美学的致敬，或者说是偏执。

My Blog

其实你现在看到的 Koslin’s Blog 是第二个版本了，我之前做过一个基于 Wordpress 的博客，后来因为对美观的追求搭建了这个 Hexo Next 驱动的博客。克服对大篇幅英语的恐惧，我尝试对 Config 文件进行了研究，小有成效，其实这个博客还算美观？

主要分享一些自己的随文散笔，偶尔有朋友的诗作，或是亲密关系者给我的信。暂时没有将这个博客大规模向外推广的计划，主要是给朋友们分享我自己的思想，寻求共鸣。
